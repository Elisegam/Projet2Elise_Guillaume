# Projet2Elise_Guillaume
----------------------------------------------------------------------------------------------
				Contact Manager
----------------------------------------------------------------------------------------------
Par Guilllaume Espitia et Elise Gamassa
le 19/11/2019


---------------------
I. Description
---------------------

Contact Manager est une application visant à gérer ses contacts.
Elle permet de visualiser tous les contacts, de les mofifier, de les suppimer et d'en créer des nouveaux.

---------------------
II. Fonctionnalités
---------------------

Fonctionnalités à la disposition de l'utilisateur :

* Lire les informations de tous les contacts
* Créer un nouveau contact
* Modifier les informations d'un contact
* Supprimer un contact

---------------------
III. Livrables
---------------------

VOus trouverez les documents suivants:

* projet.war (application sour format .war)
*projet.zip (projet Eclipse sous format .zip)
*javadoc.zip(documentation relative au projet)
*Documents(Diagramme de classes, de cas-d'utilisation et de séquence ainsi que le wireframe)

---------------------
III. Documentation:			  
---------------------
Suivre les instruction suivante:

1. Dézziper le fichier javadoc.zip
2. Rendez vous dans le dossier 'doc/' du fichier dézzipé
3. Double cliquez sur 'index.html'.

Naviguez dans toutes les classes pour lire la documentation 
respective à chacune.

--------------------------------------
IV. Execution de l'application:   
--------------------------------------
Suivre les instruction suivante:


1. Repérez le document.war;
2. Ouvrir le document contenant Wildfly 
3. cliquer sur le dossier standelone.
Une fois dans le dossier cliquez sur le dossier deployement et y glisser le fichier.war. 


3.Entrez la commande java -war Projet.war'.
